工作区/暂存区/本地库/远程库

一、用户签名：
1.适用于全局:
git config --global user.name username
git config --global user.email useremail
2.适用于当前工作目录:
git config --local user.name username
git config --local user.email useremail
3.查看用户签名：
git config user.name
git config user.email
4.删除签名：
git config --global --unset user.name

二、初始化git
git init

三、查看本地库状态
git status
注：可能会有不容易理解的分类：
changes to be committed暂存区
changes not staged for commit已修改未暂存

三、添加暂存区
git add -A/--all/filename 将文件添加至暂存区
git rm --cache filename 将文件从暂存区中删除

四、添加本地库
git commit -m "日志信息" （filename） 将文件从暂存区提交至本地库
git commit --amend -m "日志信息" 修改最后一次提交

五、历史版本
git log/reflog 查看版本信息
git reset --hard 版本号 穿越版本

六、分支操作
git branch -v 查看所有分支
git branch branchname 创建新分支（克隆当前分支）
git checkout branchname 切换分支
git merge otherbranchname 合并一个分支到当前分支

七、github远程库操作
1.远程库别名
git remote -v查看远程库别名
gir remote add 别名 远程地址
git remote remove 别名
2.远程库推送
git push 别名/远程地址 分支名
3.远程库拉取
git pull 别名/远程地址 分支名
4.远程库克隆
git clone 别名/远程地址
